## Getting Started

### Python version

- Python 3.9

### Installation

```
pipenv install
```

### Tests

```
pipenv install --dev
```

Running tests

```
pipenv run test
```

### Style format

#### Running format

```
pipenv run format
```

#### Checking format

```
pipenv run format-check
```
